A premium diamond buying, selling, and trading platform inspired by **Rally Rd.** + **Cars.com**.\

- Luxury dark theme with cyan accents\
- Rally Rd. style elegant design\
- Detailed diamond listings (Cars.com style)\

- Tailwind CSS\
- Lucide Icons\
